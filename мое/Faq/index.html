Поделиться в соц. сетях
Для всех пользователей с аккаунтом Google. Ваше приложение будет запущено в тестовом режиме. Доступ к нему смогут получить только те пользователи, которых вы добавите в список тестовых пользователей. Когда рабочая версия вашего приложения будет готова, вам может потребоваться подтвердить право собственности на него.
